☁ CloudResume

A responsive resume website deployed serverlessly on Azure. It tracks real-time visitor count and enables dynamic resume downloads through FastAPI APIs. The project demonstrates cloud automation and infrastructure-as-code principles using GitHub Actions and Azure CLI.

Cloud Architecture & DevOps Highlights

• Frontend hosted on Azure Blob Storage ($web) container with public access
• Backend hosted via Azure Web App with API endpoints for resume & metrics
• CI/CD pipelines implemented using GitHub Actions for both services
• APIs securely authenticated using Azure Service Principal
• Persistent storage using Azure Table Storage and Blob Storage
• Unit-tested codebases with Jest (frontend) and Pytest (backend)

☁ Shiptivitas

A cloud-deployable dashboard that enables teams to visually manage shipping requests. Featuring drag-and-drop columns (backlog, in-progress, complete), it supports live updates and logs every change in a PostgreSQL database.

Cloud Architecture & DevOps Highlights

• Infrastructure Provisioned via Terraform (EC2 instance, security groups, and IAM setup)
• Fully Dockerized App (Node.js backend, PostgreSQL, React frontend)
• Daily cron job cleanup (log truncation, Docker pruning) using crontab within container
• Shell-based DB verification & idempotent SQL seeding with custom seed_db.js
• Secure environment variables and SSL for DB connections
• REST API-based architecture with CORS, Express, and PostgreSQL connection pooling
• Deployed to AWS EC2 with minimal overhead and high portability
• CI/CD pipeline using GitHub Actions for image build/test/deploy
• Monitoring via Cloudwatch for EC2, S3 etc.
• Terraform state file stored on Azure Blob Storage with in-built locking

☁ StockWishlist

A cloud-native wishlist platform allowing users to track and add shares stock to their personal wishlist in real time. Built using Vue.js and AWS Lambda Function with API Gateway, it emphasizes secure authentication, seamless integration with backend APIs, and developer-friendly CI/CD.

Cloud Architecture & DevOps Highlights

• All AWS Resources Managed via Terraform, defined modular IaC for RDS (PostgreSQL), Cognito, API Gateway, Lambda, S3, Amplify — supporting full reproducibility and easy teardown
• Frontend Hosted on AWS Amplify with automatic branch-based deployments and environment separation (prod)
• Authentication via AWS Cognito, supporting secure login with Amplify Auth integration and token validation in frontend runtime
• Serverless API using AWS API Gateway + Lambda, secured with Cognito JWT verification in custom authorizers to create authenticated users pool in cognito
• PostgreSQL on AWS RDS used for persistent storage, enforced with SSL-only connections and role-based permissions
• CI/CD pipeline configured in GitHub Actions to auto-test Vue components, generate coverage reports, and deploy to Amplify
• Monitoring and Logs integrated using CloudWatch Logs, API Gateway access logs, RDS and Amplify build logs for debugging
• Local Development with Docker, enabling isolated PostgreSQL DB + API testing before deployment
• Terraform state file stored on S3 and locking provided by DynamoDB
